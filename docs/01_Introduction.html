2.2 Citations

You an include citations in your Quarto document by putting a bibliography file (e.g., Bibtex format) in the directory where the document lives and indicating its location in the bibliography: tag in the YAML. Then you can put formatted citations into the text by tagging them with the @ symbol. Datalab is a service unit within UC Davis library (Jared Joseph (2017)). By default the bibliography goes at the end of the chapter, but you can control that wil YAML settings.
